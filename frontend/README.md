# Research Co-Pilot Frontend

## Setup

1. Create `.env` from `.env.example`.
2. Set:

```env
VITE_API_BASE_URL=http://127.0.0.1:8000
```

## Run

```bash
npm install
npm run dev
```

On Windows PowerShell (if script execution is restricted), use:

```bash
npm.cmd install
npm.cmd run dev
```

## Build

```bash
npm run build
```
